# Simple Tags

[![Install (Production)](https://img.shields.io/badge/Install-Production-blue)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000028ih3YAA)
[![Install (Sandbox)](https://img.shields.io/badge/Install-Sandbox-lightblue)](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000028ih3YAA)

This package provides a flexible record tagging framework for Salesforce that allows records across multiple objects to be categorised using a structured, reportable tagging model.

Many non-profit organisations need to classify records in ways that evolve over time — for example identifying programme themes, funding areas, campaign types, supporter interests, or operational classifications. While Salesforce provides features such as Topics and Interest Tags, these are limited when it comes to reporting, data relationships, and extensibility.

This package addresses those limitations by implementing tagging using custom objects, allowing tags to behave like standard Salesforce data that can be queried, reported on, and extended.

The goal is to provide a consistent and flexible way to categorise records, helping teams better understand programmes, activities, and outcomes through improved reporting.
