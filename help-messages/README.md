# Help Messages

[![Install for Production](https://img.shields.io/badge/Install-Production-1798c1?style=for-the-badge)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001gIXpYAM)
[![Install for Sandbox](https://img.shields.io/badge/Install-Sandbox-00a1e0?style=for-the-badge)](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001gIXpYAM)

Help Messages is a lightweight guidance package for displaying contextual help directly on Salesforce records. Instead of relying on separate process notes, offline documents, or training memory, teams can surface the right message at the right point in the user journey.

The package is designed for organisations that want to guide users through record-specific steps without building full screen flows or custom page logic. A help message can apply to all records of an object or only appear when configured filter conditions match the current record.

This makes it useful for nonprofit and operational scenarios such as showing fundraising next steps on a pledged opportunity, reminding grants staff about evidence requirements at a particular stage, or prompting service-delivery teams with compliance notes when a case meets certain criteria.

Editors can draft, publish, and unpublish messages centrally, while end users only see the messages relevant to their current record and access level.
