# Report a Problem

When reporting a problem with Help Messages, include:

- the object and record type where the issue occurred
- whether the user had `Help Message Editor` or `Help Message Viewer`
- the message or filter record involved
- the expected message behaviour and the actual result
- whether the issue is about visibility, filtering, or publish status

This package is configuration-driven, so problems are often easier to diagnose when the exact message and filter setup is included.
