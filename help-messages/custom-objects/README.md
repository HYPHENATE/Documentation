# Custom Objects

Help Messages uses two main custom objects.

Together they allow teams to define contextual guidance content and the rules that decide when that content should appear on a record.

---

## Object Overview

| Object | Purpose |
|------|------|
| [Help Message](/help-messages/custom-objects/help-message) | Stores the message content, publication status, target object, and overall filter behaviour. |
| [Help Message Filter](/help-messages/custom-objects/help-message-filter) | Stores the individual filter conditions used to decide whether a message should appear. |
