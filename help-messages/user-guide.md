# User Guide

Help Messages is designed to be simple for end users. When the component is placed on a supported Lightning record page, users see a panel of messages that apply to the current record.

For most staff, the experience is read-focused. They open a record, review the displayed guidance, and use that information to decide what to do next in the process. This can include reminders, policy notes, or operational prompts.

Users with the `Help Message Editor` permission set see additional actions that allow them to open the message record and change its publication status. This supports a lightweight governance model where some users maintain the guidance while others only consume it.

Because the component reevaluates messages when relevant record fields change, users can see the guidance update as the record context changes.
