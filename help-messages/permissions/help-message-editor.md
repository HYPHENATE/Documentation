# Help Message Editor

## API Name
`Help_Message_Editor`

## Purpose

This permission set is for users who manage contextual guidance content in Salesforce.

## Access Summary

| Area | Detail |
|------|------|
| Apex access | Grants access to `HelpMessageController`. |
| Object access | Create, read, edit, and delete on `Help_Message__c` and `Help_Message_Filter__c`. |
| Field access | Editable access to the core message and filter configuration fields. |
| App access | Makes the `Help` application visible. |
| Tab access | Makes the `Help_Message__c` tab visible. |

## Typical Users

This is best suited to admins, process owners, or operational leads who maintain help content.
