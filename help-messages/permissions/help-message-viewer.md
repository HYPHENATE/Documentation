# Help Message Viewer

## API Name
`Help_Message_Viewer`

## Purpose

This permission set is for users who should see help messages on records without editing them.

## Access Summary

| Area | Detail |
|------|------|
| Apex access | Grants access to `HelpMessageController`. |
| Object access | Read access to `Help_Message__c` and `Help_Message_Filter__c`. |
| Field access | Read-only access to the message and filter fields needed for display and evaluation. |
| Editing rights | No create, edit, or delete access to the packaged objects. |

## Typical Users

This is intended for general end users who need contextual guidance during their day-to-day work.
