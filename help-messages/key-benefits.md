# Key Benefits

## Contextual User Guidance

Help Messages lets teams show practical record-level guidance directly where users are working, reducing the need to switch to separate process documents or rely on memory.

## Configuration Instead Of Code

Admins can define messages and filters using packaged objects, which makes it easier to adapt guidance as processes change without rebuilding page logic.

## Draft And Publish Workflow

The package supports draft and published states so editors can prepare, review, and release guidance safely without exposing unfinished content to all users.

## Record-Specific Relevance

Messages can apply to all records of an object or only when filter conditions match, which helps keep the guidance relevant instead of overwhelming users with generic notes.
