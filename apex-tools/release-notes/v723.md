# v7.2.3

This is a small update that introduces a new WebServiceData class, to be used with any REST web service you may be building.

When working with a REST web service (a class marked with @RestResource ) you have access to the RestContext.

As part of this, you get access to RestRequest object, the RestResponse object, which in turn gives access to the REST request body, the REST request parameters, the REST response body and REST response headers.
