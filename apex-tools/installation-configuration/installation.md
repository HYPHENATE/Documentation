# Installation Guide 

## Salesforce Requirements 
There are no Salesforce requirements for this installation, the package is designed to work in old and new environments, so if a client is on Nonprofit Cloud (NPC) or not then you can use this package. 

## Prerequisites 
None 

## Installation 
Install the package using the appropriate link below 

> [![Install (Production)](https://img.shields.io/badge/Install-Production-blue)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000029eOTYAY)

> [![Install (Sandbox)](https://img.shields.io/badge/Install-Sandbox-lightblue)](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000029eOTYAY)

## Installation Permissions
During installation select to Install for All Users 
This will ensure the Apex class included within the package is applied to all profiles
