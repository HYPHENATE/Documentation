Custom Flow LWC Components is a Salesforce package containing a broad library of Lightning Web Components designed to extend Flow.

The package README describes it as a collection of reusable components built to expand Flow capabilities and support user experience requirements that standard Flow screens do not meet on their own.

Because the package is a component library rather than a single business process app, teams typically install it as a toolkit and then selectively use the pieces that fit their implementations.
