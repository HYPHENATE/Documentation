# Relationship Centre

[![Install for Production](https://img.shields.io/badge/Install-Production-1798c1?style=for-the-badge)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001mye1YAA)

Relationship Centre is a Salesforce package for managing relationship records and related relationship-creation workflows.

The package leans heavily on packaged flows and relationship objects to support account-account, account-contact, contact-contact, and party-role relationship patterns.
