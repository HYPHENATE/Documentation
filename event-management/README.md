# Event Management

[![Install for Production](https://img.shields.io/badge/Install-Production-1798c1?style=for-the-badge)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001E713YAC)
[![Install for Sandbox](https://img.shields.io/badge/Install-Sandbox-00a1e0?style=for-the-badge)](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001E713YAC)

Event Management is a Salesforce package for managing events, registrations, tickets, favourites, and event-facing digital experiences.

The package combines custom objects, public-facing and internal Lightning components, Apex controllers, and registration flows to support the end-to-end event lifecycle from setup through attendee interaction. It is designed for teams that want to manage event operations in Salesforce while also surfacing event browsing, registration, favourites, and ticket journeys through Experience Cloud or other Salesforce-hosted digital experiences.

At a high level, teams configure events, tickets, speakers, registrations, and optional view-tracking settings in Salesforce. The package then uses Apex controllers and Lightning Web Components to render landing pages, event detail pages, favourites, registration modals, my tickets views, and e-ticket experiences.

The result is a package that covers both the operational side of event administration and the attendee-facing side of event participation, with the same underlying Salesforce records driving both.
