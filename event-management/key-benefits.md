# Key Benefits

## End-To-End Event Data Model

The package includes core objects for events, registrations, tickets, roles, favourites, and views, which gives teams a structured base for event operations.

## Public-Facing Event Experience

Packaged LWCs and Apex controllers support landing pages, event detail views, favourites, registrations, and e-ticket experiences.

## Salesforce-Native Administration

Admins and operations teams can manage event setup, reporting, and process configuration from within Salesforce rather than relying on disconnected tools.
