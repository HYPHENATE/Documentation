# v1.6.1 (March 2026)

## Enhancements
- Security & Data Access 
    - Replaced WITH SECURITY_ENFORCED with WITH USER_MODE across SOQL queries. 
    - Updated DML operations to execute using as user context. 
- Testing & Code Quality 
    - Improved overall test coverage. 
    - Test classes now implement testFor() to align with updated testing standards. 
    - API version upgraded from 66 to support the use of testFor(). 
- Accessibility 
    - Achieved WCAG 2.2 AA – 100% compliance. 
    - Achieved WCAG 2.2 AAA – 90% compliance. 
        - (Full AAA compliance is limited by Salesforce native button constraints.) 
- Standardisation 
    - Introduced dependency on ApexTools to: 
    - Support a standardised testing framework. 
    - Enable greater consistency and scalability for future enhancements. 

## Updates 
- Tag AU (Flow) 
    - Now checks whether triggers are enabled for the running user via ApexTools trigger custom settings. 
    - Supports data migrations and integration users who may use tags. 
- Flow Enhancements 
    - Added new export variable: existingTagIds (String Collection). 
    - Returns the existing Tag record IDs held in system context. 
    - Enables easy querying of previously selected tags when navigating between Flow screens. 

## Component Updates 
- tagComponent 
    - Master Label updated to SimpleTags. 
    - Description added. 
    - “View All Tags” now renders inline within the side component. 
        - (Previously used an SLDS modal, which presented usability issues at 400% zoom.) 
- tagComponentExperienceCloud 
    - Master Label updated to SimpleTags (Experience Cloud). 
    - Description added. 
    - “View All Tags” now renders inline within the side component (replacing SLDS modal for improved accessibility). 
- tagMultiSelectPicklistComponent 
    - Master Label updated to SimpleTags (Multi-Select Picklist). 
- tagObjectAvailableConfigurator 
    - Master Label updated to SimpleTags (Tag Configuration). 
    - Description added. 

## Flow Template 
- Renamed Tag LWC Flow Sample to TEMPLATE – SimpleTags LWC Flow.
