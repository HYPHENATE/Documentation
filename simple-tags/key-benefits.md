# Key Benefits

## Improved Reporting
Tags are stored as structured records, allowing them to be included in custom report types, dashboards, and analytics.

## Flexible Classification
Records can be categorised across multiple objects such as programmes, campaigns, grants, activities, or contacts.

## Structured Governance
Tags are organised into categories to help maintain clarity and consistency across teams.

## Object-Level Control
Tags can be restricted to specific objects so that users only see relevant tagging options.

## Extensible Data Model
Because the system is built using custom objects, additional metadata or relationships can easily be added if needed.
