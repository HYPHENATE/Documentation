# Dynamic Checks Administrators

`Dynamic_Checks_Administrators` is the main setup and governance permission set for the package.

## Purpose

It gives administrators access to configure checklist categories, templates, and filters, and to manage live check records in full.

## Typical Access

- full access to `Check_Template_Header__c`
- full access to `Check_Template__c`
- full access to `Check_Filter__c`
- full access to `Check__c`
- access to the package Apex classes and custom permission

## Recommended Users

Assign this permission set to Salesforce admins or platform owners responsible for checklist design and ongoing support.
