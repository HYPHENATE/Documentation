# Dynamic Checks

`Dynamic_Checks` is the main end-user permission set for the package.

## Purpose

It allows users to work with live checklist items on records without giving them full configuration control over the package.

## Typical Access

- read access to template and filter configuration needed by the component
- create and edit access to `Check__c`
- access to the package Apex classes and custom permission

## Recommended Users

Assign this permission set to staff who need to complete checks as part of day-to-day operational work on records.
