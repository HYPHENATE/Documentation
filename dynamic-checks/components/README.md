# Components

The main packaged Lightning Web Components are:

- [Dynamic Checks Container](/dynamic-checks/components/dynamic-checks-container)
- [Dynamic Checks Category](/dynamic-checks/components/dynamic-checks-category)
- [Dynamic Check](/dynamic-checks/components/dynamic-check)

The source also includes older `check`, `checksCategory`, and `checksContainer` bundles, but the primary exposed record-page component is `dynamicChecksContainer`.
