# Installation

## Salesforce Requirements

Dynamic Checks is intended to be surfaced through Lightning record pages.

## Prerequisites

- Salesforce Lightning record pages for the objects where you want the component to appear
- Admin access to add Lightning components and assign permission sets

## Dependencies

Dynamic Checks depends on [Apex Tools](/apex-tools/README) and it should be installed first.

- Apex Tools documentation: [Introduction](/apex-tools/README)
- Apex Tools production install: [Install Apex Tools (Production)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000029eOTYAY)
- Apex Tools sandbox install: [Install Apex Tools (Sandbox)](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tQB0000029eOTYAY)
