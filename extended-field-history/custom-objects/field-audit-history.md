# Field Audit History

## API Name
`FieldAuditHistory__c`

## Description

The **Field Audit History** object stores the actual audit entries generated by the package when tracked field values are created or updated.

Each row records which source record changed, which object and field were involved, the old and new values, and whether the change was captured as a created or updated value.

## Field Details

| Field | Purpose |
|------|------|
| FieldAPIName | Stores the API name of the tracked field that changed. |
| FieldLabel | Stores the user-friendly label of the tracked field at the time the audit row is created. |
| NewValue | Stores the new value after the change, converted into text form for display. |
| ObjectName | Stores the API name of the source object. |
| OldValue | Stores the previous value before the change, converted into text form for display. |
| RecordId | Stores the ID of the source record whose field changed. |
| Type | Indicates whether the audit row came from a create or update path. |
