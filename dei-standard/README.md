[![Install for Production](https://img.shields.io/badge/Install-Production-1798c1?style=for-the-badge)](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tQB000001XrOLYA0)

DEI Standard is a Salesforce package for capturing and managing diversity, equity, and inclusion data in a structured way.

The README describes it as a data-management model aligned to emerging DEI standards and intended to support eventual structured reporting outputs.

The package combines custom objects, custom metadata mappings, flows, value sets, and invocable Apex to support DEI data capture and downstream reporting preparation.
